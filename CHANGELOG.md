# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/),
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-11-26

### ✨ 新增功能
- 🎯 支持超高分辨率截图导出 (最高支持 8K 及以上)
- 🧩 实现分块渲染技术,突破 GPU 显存限制
- ⚡ 异步处理机制,防止 UI 卡死
- 📊 实时进度反馈功能
- 💧 支持文字水印和图片水印
- 🎨 支持 PNG、JPEG、WebP 三种导出格式
- 🎛️ 提供预设分辨率选项 (1080p/4K/8K)
- 🔧 支持自定义分辨率 (最高 16384×16384)
- 🎨 优雅的 Vue 3 + Element Plus UI 界面
- 🛠️ 智能阴影和相机状态恢复

### 🔧 技术实现
- 使用 Three.js 进行 3D 场景渲染
- 使用 Vue 3 Composition API 构建 UI
- 使用 Vite 作为构建工具
- 实现相机视口偏移 (setViewOffset) 进行分块渲染
- Canvas 2D 上下文进行图片拼接和水印合成

### 📝 文档
- 添加完整的 README.md
- 添加 MIT 许可证
- 添加更新日志

## [1.0.1] - 2025-12-02

### 🔧 优化改进
- 💡 **增强水印系统**: 新增9种定位选项、平铺模式、旋转角度支持
- 🛡️ **稳定性提升**: 优化分块尺寸限制(2048px),提升GPU兼容性
- ⚡ **性能优化**: 改进错误处理和资源清理机制
- 📝 **文档完善**: 更新README,添加详细的水印使用示例

### 🐛 修复
- 修复超大分辨率导出时的内存溢出问题
- 优化水印渲染性能和视觉效果
